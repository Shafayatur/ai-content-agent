# Platform Style Guide

## LinkedIn
- Character range: 800-1300 characters performs best for us.
- Format: short paragraphs (1-3 sentences), line breaks between them, no hashtags
  spam (max 3, at the end only).
- Best posting window (US audience): Tue-Thu, 8-10am ET.

## X / Twitter
- Character limit: 280 per post; use a numbered thread (2-4 posts) for anything
  longer.
- Format: punchy, one idea per post, no more than 1 hashtag per post.
- Best posting window: weekdays, 9-11am ET or 6-8pm ET.

## Instagram
- Caption length: 125-150 characters gets read in full; put the hook in the first
  line before the "more" cutoff.
- Requires an accompanying image or carousel — flag if none provided.
- Hashtags: 5-10, mixed niche + broad, placed in first comment not caption.
